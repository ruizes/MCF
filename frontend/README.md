# 医学图像分割可视化界面

## 功能

本界面提供了医学图像分割的可视化功能，支持CT和MRI图像的上传、分割和结果展示。

## 界面结构

### 输入设置
- 上传医学图像：支持.nii、.nii.gz和.h5格式的图像文件
- 图像类型选择：选择CT或MRI图像类型
- 开始分割按钮：触发图像分割请求

### 图像预览
- 原始图像：展示上传的原始医学图像
- 分割结果：展示模型分割后的结果

### 分割进度
- 加载动画：以圆圈加载方式展示分割进度
- 进度条：显示分割进度的百分比

### 最终结果
- 展示最终的分割结果图像

## 使用方法

1. 启动后端API服务器
2. 打开index.html文件
3. 上传医学图像文件
4. 选择图像类型（CT或MRI）
5. 点击“开始分割”按钮
6. 等待分割完成，查看结果

## 技术栈

- HTML5
- CSS3
- JavaScript
- Flask（后端API）
- PyTorch（深度学习模型）
- Nibabel（医学图像处理）
